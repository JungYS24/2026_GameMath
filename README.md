# 🎮 Game-Mathematics-Study
> 게임 개발의 근간이 되는 수학적 원리를 탐구하고 구현하는 실습 저장소입니다.

---

## 📅 학습 목표
- 게임 엔진(Unreal Engine) 내의 물리 및 그래픽스 수학 이해
- 벡터(Vector), 행렬(Matrix), 사원수(Quaternion)의 개념 정립 및 코드 구현
- 효율적인 게임 로직 설계를 위한 수학적 사고력 배양

---

## 🛠 개발 환경
- **Language:** C#
- **IDE:** Visual Studio 2022
- **Engine:** Unity 6
- **Version Control:** Git / GitHub

---

## 📚 주요 학습 내용

### 1. 선형대수학 (Linear Algebra)
- [ ] 벡터의 연산 (내적, 외적) 및 정규화
- [ ] 행렬 변환 (이동, 회전, 크기 조절)
- [ ] 좌표계 변환 (Local <-> World <-> View)

### 2. 삼각함수 및 기하학 (Trigonometry)
- [ ] 각도와 호도법 (Degree & Radian)
- [ ] 삼각함수를 활용한 원 운동 및 진동 구현
- [ ] 충돌 체크 알고리즘 기초

### 3. 회전과 보간 (Rotation & Interpolation)
- [ ] 오일러 각(Euler Angles)과 짐벌 락(Gimbal Lock)
- [ ] 쿼터니언(Quaternion)의 이해
- [ ] 선형 보간(Lerp)과 구면 선형 보간(Slerp)
